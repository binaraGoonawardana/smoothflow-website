Features

Drag & Drop over 500 predefined activities

Save time by using the simple drag and drop feature and arrange the elements in the work area and reorder them as required to build the workflow.

Test your logic with one click

Once you design a workflow you can test it with Smoothflow. Smoothflow will check for any logical errors going through every node in the flow and notify if there are any errors.

Draw complex workflow logic

Minimize the complexity by using the hierarchical structure available in Smoothflow. The interface facilitates easy differentiated nodes with different colors to reduce complexity.

Publish as a Go file

Publish the workflows as Go file which is a format used by Go Programming Language developed and supported by Google.

A Clear and Intuitive UI

An intuitive interface structured so that the menu provides everything you need to build workflows. This structured arrangement ensures ease of use.

 Built your own activity with a snippet

In addition to the existing activities in Smoothflow, you can create your own activities which can be used in the workflows you design. Smoothflow validates the code and saves the activities in the system.

Secure Payment Handling

A Secured Payment Gateway is integrated through CloudCharge for payment processing. Include this activity to the workflow for handling payment transactions.

Add Authentication using OAuth

Smoothflow uses OAuth, a secure authorization in a simple and standard method from web, mobile and desktop applications.

Store Data for faster Retrieval

Stores data received by the workflows in the ObjectStore to retrieve for later use.

Run your logic on IOT devices

Smoothflow can be used to build workflows which can be tiny and run on Embedded Devices and IoT Devices.

Compile your logic

The created workflows can be compiled to different platforms and downloaded as executable file which can be later used on other machines.

Publish directly to a Docker

If you do not have a dedicated server to host the workflow, Smoothflow provides you with a docker to host and run your workflows.
